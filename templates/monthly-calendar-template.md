# Monthly Editorial Calendar

## Purpose

This is a working document for planning one month of editorial content within the Latazin Editorial Operating System. It translates the quarterly plan's monthly outline into a week-by-week content schedule, specifying each planned latazin with its type, products, occasion, and objective. This is the operational layer — the document the team works from day to day.

---

## How This File Is Used in the System

```
Annual Plan  →  Quarterly Plan
                      │
                      ▼
             ┌──────────────────────────┐
             │  THIS MONTHLY CALENDAR   │  ◀── You are here
             └──────────────────────────┘
                      │
                      ▼
              Latazin Briefs  →  Latazin Generation
```

Complete one monthly calendar at the start of each month (or during the final week of the prior month). Each latazin listed here should have a corresponding latazin brief created before generation begins.
