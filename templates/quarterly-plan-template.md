# Quarterly Editorial Plan

## Purpose

This is a working document for planning a single quarter of editorial operations within the Latazin Editorial Operating System. It translates the annual editorial plan's quarterly arc into actionable priorities: what to publish, when, for whom, around which products and occasions, and using which latazin types. The output feeds directly into monthly calendar creation.

---

## How This File Is Used in the System

```
Annual Editorial Plan
         │
         ▼
  ┌──────────────────────────┐
  │  THIS QUARTERLY PLAN     │  ◀── You are here
  └──────────────────────────┘
         │
         ▼
  Monthly Calendars  →  Latazin Briefs  →  Latazin Generation
```

Complete one quarterly plan per quarter. Begin planning 2–4 weeks before the quarter starts. Reference the annual plan's seasonal arc, product emphasis, and audience priorities for this quarter.
