# sterling-stitches

Sterling Stitches is a luxury menswear brand based in Abuja, Nigeria. Tagline: **"Perfection in every stitch."** The brand specialises in contemporary Nigerian menswear — ready-to-wear garments and accessories that blend Yoruba textile heritage (Adire, Ankara, Aso-oke) with modern tailoring. Primary markets: Abuja, Lagos, and the Nigerian diaspora.
